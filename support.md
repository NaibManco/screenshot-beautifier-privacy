# Support — Screenshot Beautifier

If you need help, please use one of the following channels:

- **GitHub Issues (recommended):** https://github.com/NaibManco/screenshot-beautifier/issues

---

## What to include in a support request
- Your Chrome version
- Extension version (from `chrome://extensions/`)
- What you expected vs what happened
- Screenshots (optional)

> Note: Screenshot Beautifier processes images locally. Please avoid sharing sensitive screenshots publicly.
